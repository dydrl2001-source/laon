# agents/ — 도메인 전문 에이전트 정의

특정 도메인 작업 시 반복 호출하는 **전문가 에이전트의 정의 카드**. 각 카드는 시스템 프롬프트 템플릿 + 트리거 + 입출력 규약을 담음.

## 두 가지 사용 방법

**방법 1 — 참조 문서로 활용 (현재 디폴트)**
- 메인 Claude가 Agent 도구로 subagent 호출 시 본 문서를 시스템 프롬프트의 토대로 사용
- 별도 등록 불필요, 작성한 즉시 바로 활용 가능
- 본 디렉토리 = 매니저 거버넌스의 일부 (git 추적)

**방법 2 — Claude Code 글로벌 에이전트로 정식 등록 (선택)**
- 본 폴더 → `~/.claude/agents/` 심볼릭 링크 또는 복사
- 자동 인식되어 Agent 도구 `subagent_type`에 노출
- 추후 필요 시 `playbooks/에이전트-등록절차.md` 추가 (Phase 7)

## 작성 규약

각 에이전트 파일은 다음 6요소 포함:
1. **이름·간단 소개**
2. **트리거** (언제 활성화)
3. **입력** (필수·선택)
4. **시스템 프롬프트** (역할·제약·출력 형식)
5. **출력 형식**
6. **실패 처리** (잘못된 입력·외부 자원 부재 시)
